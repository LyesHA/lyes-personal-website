Contenus du cours






Principaux thèmes traités (contenus)

Voici la liste des grands titres des contenus à traiter essentiellement dans ce cours



  • Comprendre l'Internet et le World Wide Web

    Aperçu de l'Internet, concepts de base du réseau client-serveur, réseau local (LAN), Réseau étendu (WAN), navigateurs Web, serveurs Web, localisateur uniforme de ressource (URL), Service d'hébergement Web.

  • Création de pages Web avec HTML

    Introduction, Web Design, HTML / XHTML, normes de codage du W3C, HTML: structure d'un fichier HTML, balises HTML, éléments structuraux, éléments de contenu, attributs, valeurs, tableaux, listes, lien text, lien image cartes d'image, lien email, liste de navigation, mise en page web, multimédia, nouvelles fonctionnalités de HTML.

  • Utilisation des feuilles de style en cascade dans une page Web

    Introduction à CSS1, CSS2 et CSS3, types de CSS: style en ligne, interne et externe, CSS: sélecteurs, couleur, styles pour les listes, modèle de boite (marges, rembourrage, fonds et bordures), formatage de tableaux avec CSS, effets de texte, transformations 2D/3D, animations, mise en page.

  • Intégration de scripts côté client dans une page Web

    Introduction à DHTML, notions de base de JavaScript : types de données, variables, constantes, opérateurs et expressions, structures de contrôle, tableaux, fonctions et méthodes intégrées, fonctions utilisateurs; JavaScript en ligne, interne et externe, fenêtres et cadres, formulaires et expressions régulières, cookies, JavaScript Orienté objet.

  • Programmation BOM Browser Object Model

    Introduction à BOM, objet Window, objet Navigator, objet Screen, object History et location

  • Programmation avec DOM (le Document Object Model)

    Introduction aux DOM, DOM HTML, éléments, attributs, noeuds, propriétés méthodes, accès et modification d'éléments HTML, événements et navigation DOM.

  • Utilisation de jQuery

    Introduction à jQuery, Sélection d'éléments, gestion des événements, manipulation du style et de l'animation, manipulation du DOM.
